# How to use
```diff
+ Download / Clone Project
+ 🕹 cd to the project folder
+ 🕹 Run: npm i
+ Go to the Firebase console and create a project
+ Enter your Firebase keys and ids in .env 
+ 🕹 Run: npm start
+ voilà 🤌🏼
```

Enjoy 🚀
